# apkbazaar
Get apk from cafebazaar store

# Usage
```
usage: bazaarapk [-h] -p P -n N

Download apk from bazar

optional arguments:
  -h, --help  show this help message and exit
  -p P        packagename
  -n N        apk name

python bazaarapk.py -p 'xxx.xxxxx' -n 'apkname'

```
# Requirements

```
pip install requests
```
